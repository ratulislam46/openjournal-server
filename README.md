# 📝 Open Journal Server

**Open Journal** is a modern and feature-rich backend system for a blogging platform. Built with Node.js and Express.js, it utilizes MongoDB as its database. This server efficiently handles Blog CRUD operations, Wishlist management, and a real-time Comment system.

---

## 🚀 Key Features

- **Blog CRUD:** Full functionality to Create, Read, Update, and Delete blog posts.
- **Smart Search:** Case-insensitive search filtering based on blog titles.
- **Feature Sorting:** Logic to sort "Featured Blogs" based on the length of the blog description.
- **Wishlist Integration:** Allows users to save their favorite blogs for later reading.
- **Interaction System:** A dedicated comment section for users to express their thoughts on each blog.
- **Security:** Database credentials are kept secure using environment variables (dotenv).

---

## 🛠️ Tech Stack

- **Server:** Node.js
- **Framework:** Express.js
- **Database:** MongoDB Atlas
- **Security:** Dotenv, CORS
- **ID Generation:** ObjectId (MongoDB)
